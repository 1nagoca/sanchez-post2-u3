# sanchez-post2-u3

Laboratorio: Dashboard con Grid y Flexbox

# descripción

Esta práctica se centra en la implementación de Layouts avanzados y Diseño Responsivo para las secciones de Servicios y Proyectos del portafolio. Se aplicó CSS Grid para gestionar la rejilla de proyectos y Flexbox para la alineación flexible de los servicios, garantizando una estructura moderna y organizada. Además, se integraron Media Queries para asegurar que la interfaz sea totalmente adaptable (responsive), optimizando la visualización en dispositivos móviles y de escritorio, y cumpliendo con los estándares de accesibilidad y usabilidad web
